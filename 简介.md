# 实时新闻真伪辨别工具 — 项目大纲

## 一、项目概述

基于大语言模型（LLM）的实时新闻真伪辨别工具。用户输入新闻文本或URL，系统自动搜索网络、筛选信源、交叉验证、加权评分，流式输出分析结果。

**三大核心原则**：
1. **信息充分**：分析前从网络获取足够相关信息，不依赖模型内部知识
2. **去伪存真**：对网络信息进行A/B/C/D四级信源评级，交叉验证剔除虚假信息
3. **公正客观**：正反论据均衡呈现，标注信源立场倾向，让用户自行判断

## 二、核心功能模块（均已实现）

### 1. 新闻采集

- **文本输入**：粘贴新闻文本直接分析
- **URL 抓取**：粘贴链接自动抓取正文（httpx + readability + Playwright JS渲染兜底）
- **浏览器插件**：Edge/Chrome 扩展，选中页面文字一键获取

### 2. 网络信息检索

- **SerpAPI（Google）搜索**：中文 + 英文关键词并发检索
- **检索关键词自动生成**：LLM 提取多个维度（核心主张、参与方、权威声明、质疑、时间地点）
- **正文提取**：静态抓取 → Jina Reader → Playwright 三级兜底

### 3. 信息可信度筛选

- **信源 A/B/C/D 四级评级**：100+ 已知域名规则 + 域名特征推断
- **多源交叉验证**：每条论据标注是否被 ≥2 个独立信源确认
- **虚假信号识别**：标题党、情绪煽动、信源缺失、逻辑矛盾、利益导向
- **传播路径追踪**：首发来源 + 跨平台传播时间线

### 4. AI 分析引擎

- **DeepSeek V4-Flash**（主力）/ Anthropic Claude（备用）/ OpenAI 兼容接口
- **核心主张加权评分**：AI 提炼 3-5 条核心主张，分配权重，加权平均得出总概率
- **结构化输出**：来龙去脉 + 摘要 + 真实概率 + 正反论据 + 论证过程
- **温度 0.0**：确定性输出，同输入同结果

### 5. 结果展示

- **圆形概率仪表盘**：颜色从红（假）到绿（真），右侧并排核心主张进度条
- **论据卡片**：每条标注立场、信源等级、交叉验证状态、来源链接
- **传播路径时间线**：首发 → 转发节点
- **可疑信号红色预警框**
- **流式分析进度**：搜索 → 提取 → 分析 → 结果，渐进展示
- **下载报告**：一键导出 TXT 格式分析报告

### 6. 数据持久化

- **分析历史记录**：SQLite 存储，下拉框选择回顾
- **谣言数据库**：已知谣言快速匹配，虚假内容自动入库
- **结果缓存**：同输入 5 分钟内复用，避免重复请求

## 三、技术架构

```text
┌──────────────┐    ┌──────────────┐    ┌───────────────┐
│  Gradio Web  │───>│  Python 后端  │───>│  DeepSeek API  │
│  (响应式UI)   │<───│  (分析管线)   │<───│  (V4-Flash)    │
└──────────────┘    └──────┬───────┘    └───────────────┘
                           │
                     ┌─────┴──────┐
                     │  SerpAPI    │
                     │  (Google搜索)│
                     └────────────┘
```

- **前端**：Gradio 5.x（Python 原生，PC/移动端响应式）
- **后端**：Python 3.13 + FastAPI
- **AI 服务**：DeepSeek V4-Flash（OpenAI 兼容接口），支持 Anthropic/OpenAI 切换
- **搜索**：SerpAPI（Google）
- **网页抓取**：httpx + readability + Jina Reader + Playwright 四级兜底
- **数据**：SQLite（历史记录 + 谣言库 + 缓存）

## 四、输出内容规范

| 字段 | 说明 |
| ---- | ---- |
| 来龙去脉 | 事件前因后果与完整背景 |
| 新闻摘要 | 核心事实主张 |
| 真实概率 | 核心主张加权平均，0-100% |
| 论据列表 | 正反论据，每条标注立场、信源等级、交叉验证状态、来源链接 |
| 论证过程 | 逻辑推理链，从论据到结论 |
